outputImage()

outputImage(array $image, string $class_name, string $alt, array $sizes, bool $lazy_load)

Output an image tag.

Parameters

array $image

An array of image attributes like that from ACF (http://www.advancedcustomfields.com/resources/field-types/image/)
string $class_name

Class to apply to image
string $alt

Optional alt text for image. If not provided, attempts to get alt text from WordPress or uses image name
array $sizes

Optional array of pre-defined sizes to try to use (for retina images)
bool $lazy_load

If TRUE, outputs image tag with data- attributes instead actual image attributes so that the image can be lazy loaded. Defaults to FALSE.
