updatePageColumnsInAdmin(array $columns) : array

Add additional columns to the Page list in the WP Admin.

By default, adds a column for Template.

Called via manage_edit-page_columns WP Action.

Calls plinth_page_columns filter with $columns to allow child theme to edit column list.

Based on: http://www.elliotcondon.com/advanced-custom-fields-admin-custom-columns/

Parameters

array $columns

Array of current columns

array —

Modified columns array
